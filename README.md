# 📈 Pakistan's Economic Pulse: DataFest

A data-driven deep dive into Pakistan's macroeconomic landscape, focusing on high-impact forecasting and multi-sectoral correlation analysis. This project was developed for the **PBS DataFest**.

## 🚀 Project Overview
This project goes beyond static reporting to forecast economic trends by integrating non-traditional data drivers like **Climate** and **Trade** with industrial output markers (**QIM**). 

### Key Objectives
*   **Time-Series Forecasting:** Leveraging SARIMAX models to predict economic trajectories.
*   **Multi-Sector Correlation:** Analyzing how climate shifts and trade fluctuations impact industrial growth patterns.
*   **Interactive Insights:** Visualizing complex economic dependencies for actionable policy or investment intelligence.

## 🛠️ Technical Stack
*   **Language:** Python (Pandas, NumPy)
*   **Forecasting Model:** SARIMAX (Seasonal Auto-Regressive Integrated Moving Average with Exogenous Factors)
*   **Visualization:** Power BI (Interactive Dashboards)
*   **Data Domains:** 
    *   **QIM:** Quantum Index of Manufacturing (Industrial output)
    *   **Climate:** Historical weather patterns and their impact on productivity.
    *   **Trade:** Import/Export volumes and trade balance data.

## 📊 Methodology
1.  **Data Integration:** Merging disparate datasets from PBS and climate repositories.
2.  **Exogenous Variable Selection:** Using Trade and Climate data as exogenous features in the SARIMAX model to capture external shocks.
3.  **Trend Analysis:** Visualizing industrial growth patterns against global and local economic shifts.
